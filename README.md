# RoomFood

RoomFood is a website for managing meal planning in shared households.

## Description

RoomFood provides capabilities for users to create rooms, in which they are able to plan meals. It is possible to invite
other users to the created rooms, and the planned meals are shared between all members.

When creating a meal, a user can specify which recipes are to be used in the preparation of the meal, who is responsible
for cooking it and show how many people will be eating it. After creating a meal, all members of the room will also be 
able to see which ingredients are needed for future planned meals and which are already available in the room.

There is also a feature that allows you to see all meals the user is responsible for cooking in all the rooms he is part
of.

## Features (in development)
- Profile system
- Room meal planning
- Recipe creation and sharing
- Automatic grocery shopping list
